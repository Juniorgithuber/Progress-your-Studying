# دفتر مطالعه (Study Tracker)

یک اپلیکیشن اندروید ساده و آفلاین برای ثبت زمان مطالعه، پارت‌های ۳۰ دقیقه‌ای، و زمان استراحت — با امکان اتصال اختیاری به Notion، آلارم واقعی گوشی، تقویم اندروید، و تحلیل هوشمند هفتگی با Claude API.

## حریم خصوصی

- تمام داده‌های مطالعه (تایمر، پارت‌ها، درس‌ها) فقط روی خود گوشی ذخیره می‌شن (SharedPreferences)، نه در هیچ سروری.
- اتصال به Notion و Claude API کاملاً اختیاریه؛ اگه توکن/کلید رو در تنظیمات اپ خالی بذاری، هیچ داده‌ای جایی ارسال نمی‌شه.
- هر کاربر توکن/کلید خودش رو وارد می‌کنه؛ هیچ کلیدی داخل کد این ریپو نیست.
- کد این اپ کاملاً عمومی و قابل بررسیه — هرکسی می‌تونه قبل از نصب، کل کد رو در همین ریپو بخونه.

## دریافت اپ (بدون نیاز به نصب Android Studio)

APK آماده همیشه اینجاست:
**Releases → آخرین نسخه (tag: `latest`) → فایل `study-tracker.apk`**

هر بار که کد این ریپو آپدیت بشه، GitHub به‌صورت خودکار (نه از روی گوشی یا کامپیوتر شخصی کسی) نسخه‌ی جدید رو می‌سازه و در همون بخش Release جایگزین می‌کنه.

## ساخت دستی (اختیاری)

اگه خواستی خودت از روی کد بسازیش:
```
git clone <آدرس این ریپو>
cd StudyTracker
./gradlew assembleDebug
```
فایل خروجی در `app/build/outputs/apk/debug/app-debug.apk` قرار می‌گیره.
